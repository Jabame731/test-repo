Use this audit to merge multiple records that were created for a single contact. Using the “Merge Record” option will merge the contacts to a single record, and you will be required to select which record will be merged into, and which record will be deleted. Using the “Mark as same person” button will leave the records separate for the sales users, but allow Managers to view a single record for each contact.
